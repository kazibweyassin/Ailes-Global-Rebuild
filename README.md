
# AILES Global - Premium Study Abroad & Scholarship Consulting Platform

A modern, premium website for AILES Global, helping African students (especially women) access world-class education opportunities globally.

## Features

### Core Features
- **AI-Powered University Matcher** - Personalized university recommendations based on student profiles
- **Enhanced Scholarship Finder** - Dynamic search with advanced filters
- **Student Dashboard** - Comprehensive portal for tracking applications (coming soon)
- **Application Status Tracker** - Visual timeline of application progress (coming soon)
- **AI Scholarship Assistant** - Chat interface for instant guidance (coming soon)

### Pages
- ✅ Homepage with hero section and key sections
- ✅ About Us - Brand story and values
- ✅ Services - Comprehensive service offerings
- ✅ Scholarships - Dynamic searchable database
- ✅ Success Stories - Student testimonials and case studies
- ✅ Blog/Resources - Educational articles and guides
- ✅ Pricing - Transparent package options
- ✅ Contact - Contact form and information
- ✅ University Matcher - AI-powered matching tool

## 🛠️ Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** TailwindCSS
- **UI Components:** ShadCN UI
- **Language:** TypeScript
- **Icons:** Lucide React
- **Animations:** Framer Motion (ready to use)

## 📦 Installation

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run development server:**
   ```bash
   npm run dev
   ```

3. **Open in browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)

## 🔧 Configuration

### Environment Variables

Create a `.env.local` file for environment variables (when needed):

```env
# Database (for future features)
DATABASE_URL=""

# NextAuth (for authentication)
NEXTAUTH_URL="http://localhost:3000"
NEXTAUTH_SECRET=""

# AI Services (for AI features)
OPENAI_API_KEY=""
```
